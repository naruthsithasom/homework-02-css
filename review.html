1. CSS syntax ประกอบด้วยอะไรบ้าง Ex. Code
p{ color: blue; font-size: 16px; }
Selector Property Value
p

selector หมายถึงให้ใช้ <p> นี้

 color font-size

property หมายถึงการใช้คุณลักษะ และในที่นี้คือ สี, ขนาดตัวอักษร

 blue 16px

Value หมายถึงกำหนคให้อักษรแสดงสี blue และให้ขนาดอักษรเป็น 16px

2. การเขียน CSS มีกี่แบบอะไรบ้าง Ex. Code

<body>

<style>

p{color:blue;font-size:16px;}

</style>

</body>

<head>

link rel="stylesheet" href="css/style.css"

</head>

<body>

<p style="color:blue;font-size:16px;"> Text </p>

</body>
Internal CSS External CSS Inline CSS
แบบที่ 1 (Internal CSS) คือการนำtag <style> ไว้ใน <body> แบบที่ 2 (External CSS) คือการนำไฟล์ .css แยกไว้นอกโฟลเดอร์ และะต้องใส่ไว้ใน tag "link" ซึ่งจะอยู่ใน <haed> แบบที่ 3 (Inline CSS) คือการเขียน stylecss ไว้ใน element นั้นๆ
