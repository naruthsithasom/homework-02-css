1. CSS syntax ประกอบด้วยอะไรบ้าง

Ex. Code
p{ color: blue; font-size: 16px; }
Selector Property Value
p

หมายถึงให้ใช้ selector นี้
color font-size

หมายถึงการใช้คุณลักษะproperty และในที่นี้คือ สีและขนาดตัวอักษร
blue 16px

หมายถึงกำหนคให้อักษรแสดงสี blue และให้ขนาดอักษรเป็น 16px

2. การเขียน CSS มีกี่แบบอะไรบ้าง

Ex. Code

<body>

<style>

p{color:blue;font-size:16px;}

</style>

</body>

<head>

link rel="stylesheet" href="css/style.css"

</head>

<body>

<p style="color:blue;font-size:16px;"> Text </p>

</body>
